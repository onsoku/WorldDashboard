# World Dashboard v1.0

**自分で作る辞典** — 指定したテーマについてAIエージェントがWeb検索と学術論文検索を行い、結果をインタラクティブなダッシュボードに表示します。

[English](../README.md) | [中文](./README.zh.md) | [Español](./README.es.md) | [Italiano](./README.it.md) | [Français](./README.fr.md)

## 特徴

- **AI調査エージェント** — Claude CodeがWebと学術データベース(Semantic Scholar)を自動検索
- **インタラクティブダッシュボード** — 概要、キーワードTreemap、ソース一覧をタブ切替
- **落合陽一式論文要約** — 各論文に6項目の構造化要約（どんなもの？/ 新規性 / 手法 / 検証 / 議論 / 次に読むべき論文）
- **論文優先表示** — 学術論文タブがデフォルト、引用数順ソート
- **多言語対応** — UI・生成コンテンツが英語、日本語、中国語、スペイン語、イタリア語、フランス語に対応
- **テーマ切替** — ライト、ダーク、モノクロの3テーマ
- **設定永続化** — テーマ・言語設定をlocalStorageに保存
- **成長するナレッジベース** — 各トピックをローカルに保存、いつでも参照可能
- **ドリルダウンナビゲーション** — Treemapのキーワード、概要の重要知見、「次に読むべき論文」をクリックすると、そのテーマで即座に新しい調査を開始（`meta.parentSlug`で親子関係を追跡）
- **インポート/エクスポート** — ヘッダーの↓ボタンで現在のトピックをJSONとしてエクスポート、サイドバーの↑ボタンまたはJSONファイルのドラッグ&ドロップでインポート
- **柔軟なスキーマ** — `meta.topic`と`meta.slug`のみ必須、他のフィールドはすべて任意。テーマ別データ用の拡張システム（地図、タイムライン、表、チャート、プロフィール）
- **リッチ表示** — 概要と論文要約でMarkdownレンダリング（表、太字、見出し、リスト）。表・チャート（棒/折れ線/円グラフ、Recharts使用）・タイムライン用の拡張レンダラー
- **バージョン管理付き更新** — 既存トピックを最新情報で更新しつつ、バージョン履歴を保持。過去のバージョンはそのまま閲覧可能。事実が変わった場合の修正も追跡
- **文化対応翻訳** — 6つの対応言語にトピックを翻訳。文化的差異を自動評価し、単純な逐語訳を超えた翻訳を実現。文化的文脈が異なる場合は補足Web検索を実施
- **拡張チャート＆マップ** — 7種類のチャート（棒、折れ線、円、面、レーダー、散布図、積み上げ棒）をマルチシリーズ対応。Leaflet/OpenStreetMapによるマーカー・ポップアップ付きインタラクティブ地図
- **構造化された辞典ページ** — テーマの性質に応じた構成計画に基づき、理解・位置づけ・次の行動の3観点を満たす辞典ページを自動生成
- **並列ジョブ実行** — 最大3つの調査・翻訳ジョブを同時実行可能。各ジョブの進捗をリアルタイム表示
- **IME対応** — トピック入力欄で日本語・中国語などの漢字変換中にEnterキーで誤送信しない
- **JSON自動修復** — 生成されたJSONデータのバリデーションと自動修復機能

## 技術スタック

| レイヤー | 技術 |
|---------|------|
| フロントエンド | React 18 + TypeScript + Vite 6 |
| スタイリング | Tailwind CSS 4 |
| Markdown | react-markdown + remark-gfm |
| チャート | Recharts |
| 地図 | Leaflet + react-leaflet |
| アイコン | Lucide React |
| AIエージェント | Claude Code CLI |
| 論文検索 | Semantic Scholar API (無料) |
| データ | ローカルJSONファイル |

## 前提条件

- [Node.js](https://nodejs.org/) 18以上
- [Claude Code CLI](https://www.npmjs.com/package/@anthropic-ai/claude-code) (`npm install -g @anthropic-ai/claude-code`)
- Claude Codeサブスクリプション

## セットアップ

```bash
git clone git@github.com:onsoku/WorldDashboard.git
cd WorldDashboard
npm install
claude auth login   # 初回のみ
npm run dev
```

ブラウザで http://localhost:5173 を開く

## 使い方

1. サイドバーの **「+ 新規」** をクリック
2. 調査テーマを入力
3. **「調査を開始」** をクリック — 進捗がリアルタイム表示
4. 完了後、ダッシュボードに結果が表示される（概要、キーワード、ソース）
5. サイドバーから過去のトピックに切替可能
6. **ドリルダウン** — キーワードや提案をクリックして関連トピックを調査
7. **エクスポート** — ヘッダーのダウンロードアイコンでJSONダウンロード、**インポート** — サイドバーのアップロードアイコンまたはJSONファイルをドラッグ&ドロップ
8. **更新** — 更新アイコンをクリックしてトピックを最新情報にリフレッシュ
9. **翻訳** — 翻訳アイコンをクリックしてトピックを別の言語に変換
10. **並列実行** — 前のジョブの完了を待たずに複数の調査を開始可能
11. 設定アイコンからテーマ・言語を変更
